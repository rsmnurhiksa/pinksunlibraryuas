# 📚 LibraryHub

LibraryHub adalah aplikasi Web Full-Stack sederhana untuk manajemen perpustakaan yang dibuat sebagai proyek Ujian Akhir Semester (UAS) mata kuliah Pemrograman Web 2.

## ✨ Fitur

- Login & Logout
- Dashboard
- CRUD Kategori
- CRUD Buku
- Relasi Buku dengan Kategori
- Pencarian Data Buku

## 🛠️ Teknologi

- PHP Native
- MySQL
- Bootstrap 5
- HTML
- CSS
- JavaScript

## 🗄️ Database

Nama Database:

libraryhub

Import file:

database/libraryhub.sql

## 🚀 Cara Menjalankan

1. Install XAMPP.
2. Aktifkan Apache dan MySQL.
3. Salin project ke folder `htdocs`.
4. Import `libraryhub.sql` ke phpMyAdmin.
5. Atur koneksi database pada `config/koneksi.php`.
6. Buka browser dan akses:

http://localhost/libraryhub
